## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/seanohtani-dotcom/DOAX.git
cd DOAX

# Install dependencies
npm install
cd backend && npm install && cd ..

# Start backend
cd backend && npm start

# Open public/index.html with Live Server
```

**📚 Full Guide:** [docs/QUICK-START.md](docs/QUICK-START.md)

## ✨ Features

- 🎬 **TypeScript Video Player** - Error handling, keyboard controls, cross-browser
- 🛒 **Smart Shopping Cart** - Type-safe, localStorage persistence, validation
- 💖 **Wishlist System** - Save favorites, sync across sessions
- 💳 **Checkout Flow** - 3-step process with validation
- 🎨 **Premium Design** - Glassmorphism, animations, 3D effects
- 📱 **Responsive** - Mobile, tablet, desktop optimized
- ♿ **Accessible** - WCAG compliant, keyboard navigation
- ⚡ **Fast** - 60fps animations, optimized performance

## 🌐 API Endpoints

```
GET    /api/products       # Get all products
GET    /api/products/:id   # Get product by ID
POST   /api/wishlist/add   # Add to wishlist
DELETE /api/wishlist/:id   # Remove from wishlist
POST   /api/orders         # Create order
```
